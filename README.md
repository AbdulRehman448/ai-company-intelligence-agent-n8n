# 🚀 AI Company Intelligence Agent (n8n)

> Research any company in minutes using AI.

A production-ready AI automation workflow built with **n8n** that transforms a company website into a comprehensive executive business intelligence report.

Instead of spending 30–60 minutes manually researching a company, this workflow automatically collects public information, analyzes it with AI, generates a professional PDF report, and emails it to the user.

# 🚀 Why This Workflow?

Companies spend valuable time researching prospects before meetings.

This workflow automates the entire process by combining multiple AI services into one intelligent pipeline, helping teams make faster and more informed decisions.

---

# 🔄 Workflow Steps

1. Validate company URL
2. Scrape website using Firecrawl
3. Research company with Exa AI
4. Fetch latest news
5. Merge all collected information
6. Generate AI executive report using Gemini
7. Convert report to PDF
8. Email the report automatically
